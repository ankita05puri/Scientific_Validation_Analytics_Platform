# Scientific Requirements

## Accuracy

- Compare candidate specimen type against reference specimen type.
- Calculate difference and % bias.
- Calculate correlation and regression.
- Evaluate agreement with reference method.

## Precision

- Intra-assay precision.
- Inter-assay precision.
- Mean.
- Standard deviation (SD).
- Coefficient of variation (CV%).

## Bias

- Difference = Candidate - Reference.
- % Bias = ((Candidate - Reference) / Reference) x 100.
- Mean bias.
- Median bias.

## Correlation

- Pearson r.
- R-squared (R2).
- Scatter plots.

## Regression

- Slope.
- Intercept.
- Equation.
- Method comparison.

## Method Comparison

- Serum vs Microtainer.
- Serum vs DBS.
- AU480 vs Reference Method.
- Access 2 vs Reference Method.

## Validation Studies

- Accuracy.
- Precision.
- Linearity.
- Recovery.
- Reference range verification.
- Reportable range verification.
